# 🐔 Chicken Breed Scanner (Frontend)

A React.js Web App for Identifying Chicken Breeds Using Image Upload

--- 

## Features

- Upload your own chicken image for scanning.  
- Recommended sample images with a **click-to-popup preview**.  
- User guide on the right side explaining steps.  

---

## Installation

1. Clone Repository:

---bash / terminal
git clone https://github.com/yourusername/chicken-breed-scanner.git
cd chicken-breed-scanner


2. Install dependencies:

npm install

3. Start the development server:

npm start

---
